█ ⚠️ IMPORTANTE - SEGURANÇA
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

As senhas estão em TEXTO PURO no banco!

Para produção, você DEVE implementar:
- Hash de senhas (BCrypt, Argon2)
- Salt
- Política de senhas fortes
- Limite de tentativas de login
- Timeout de sessão

Exemplo:
```kotlin
// Adicione ao build.gradle.kts:
implementation("org.mindrot:jbcrypt:0.4")

// Use para hash:
val hashed = BCrypt.hashpw(senha, BCrypt.gensalt())

// Use para validar:
val valido = BCrypt.checkpw(senha, hashSalvo)
```
